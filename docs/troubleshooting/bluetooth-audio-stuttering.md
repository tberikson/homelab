Problem:
Bluetooth audio stuttering on my Arch Linux desktop.

Symptoms:
Audio dropouts every 10-20 seconds when using wireless bluetooth headphones.

Investigation:
- Checked Pipewire logs
- Verified codec negotiation
- Examined motherboard Bluetooth antenna configuration

Root Cause:
Motherboard Wi-Fi/Bluetooth antenna was disconnected, reducing signal quality.

Resolution:
Installed antenna and verified stable connectivity.
